# tic-tac-toe (Python)

## Project Description:
This is a terminal-based Tic-Tac-Toe game implemented in Python. The game supports a two-player mode as well as a player vs computer mode. Players can choose their own symbols, their display name, and the game tracks score and game history across multiple rounds.

Additional features of this code include: 
1) A scoreboard tracking wins and draws, and declaring the overall winner
2) Game history across multiple rounds is tracked
3) The type of the win, and the winning cells, are highlighted for simplicity
4) A random computer mode
5) Allows multiple games to be played in a single session

## How to Run the Program:
1) Kindly make sure that Python is installed
2) Download or clone this repository
3) Please open a terminal in the project folder
4) Kindly execute the following command:
```bash
python tic-tac-toe.py
```
5) Follow the in-game commands as and when they appear on the terminal
